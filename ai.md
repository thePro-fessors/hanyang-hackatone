Listed directory hanyang

해커톤 제출용 **개인 기억 비서 웹앱** 프로젝트에 딱 맞는 기술 스택을 제안해 드립니다. 

해커톤은 제한된 시간 내에 완성도 높은 **MVP(최소 기능 제품)**와 **수려한 UI/UX(WOW 포인트)**를 보여주는 것이 핵심입니다. 이를 위해 **Next.js + TypeScript** 조합을 강력히 추천합니다.

---

### 🛠️ 추천 기술 스택

1. **프레임워크 & 언어: Next.js (React) + TypeScript**
   - **이유**: 프론트엔드(UI)와 백엔드 API(Gemini AI 호출 및 DB 연동)를 **TypeScript 한 언어**로 같은 프로젝트에서 동시에 개발할 수 있어 개발 속도가 가장 빠릅니다. Vercel을 통해 1분 만에 실시간 웹 배포가 가능하여 해커톤 발표 시 시연하기 매우 유리합니다.
2. **AI API: Gemini 1.5 Flash (Google Gen AI SDK)**
   - **이유**: 사용자가 올린 사진(장소, 칠판, 명함 등)을 분석하여 **장소, 상세 내용, 핵심 키워드**를 자동으로 구조화된 JSON 데이터로 변환해 주는 멀티모달 능력이 매우 뛰어납니다. 반응 속도도 매우 빨라 실시간 앱에 적합합니다.
3. **데이터베이스: Supabase (PostgreSQL + pgvector)**
   - **이유**: 가입과 설정이 5분 만에 끝나며, 단순 키워드 검색을 넘어 **"의미 기반 검색(Semantic Search)"**을 구현할 수 있습니다. 예를 들어 사용자가 "수행평가 언제였지?"라고 물었을 때, AI가 입력된 질문의 의미를 이해하고 관련 사진과 기억을 정확히 찾아줍니다.
4. **스타일링: Vanilla CSS (혹은 원하시는 경우 Tailwind CSS)**
   - **이유**: 유리 질감(Glassmorphism) 효과, 다크 모드, 카메라 촬영 시 미세한 마이크로 애니메이션 등 해커톤 심사위원을 사로잡을 감각적이고 트렌디한 디자인을 자유롭게 구현할 수 있습니다.

---

### 💡 해커톤용 핵심 기능 시나리오
1. **기억 업로드 (카메라/갤러리)**: 모바일/웹 카메라로 사진을 찍거나 이미지를 업로드하면, 화면에 멋진 분석 애니메이션이 나오며 AI가 기억 데이터(장소, 일시, 내용, 태그)를 자동 생성.
2. **타임라인/대시보드**: 저장된 기억들이 트렌디한 그리드/카드 형태로 정렬되며, AI가 추출한 키워드로 자동 분류됨.
3. **AI 기억 검색 & 채팅**: "아까 본 과학 공지사항 언제까지 내랬지?" 같은 질문을 입력하면, AI가 관련 사진을 띄워주고 직접 답변까지 생성하여 보여줌.
