## BridgeAI - 한국 생활을 이해시키는 AI 생활 안내서
### 한 문장 정의
한국의 복잡한 행정 문서와 생활 규칙을 AI가 분석하여, 누구나 이해하기 쉬운 언어로 설명하고 필요한 행동까지 안내하는 생활 정보 도우미
한국 생활 정보 부족 -> AI가 제도와 문맥 설명
## 핵심기능
### 1. 문서 이해 AI
업로드: 건강보험료 납부 안내문
AI:
```
이 문서는 건강보험료를 내라는 안내입니다.

해야 할 일:
✅ 8월 10일까지 납부

놓치면:
⚠️ 연체료가 발생할 수 있습니다.

방법:
1. 은행 앱 접속
2. 보험료 메뉴 선택
3. 납부 진행
```
### 2. 한국 법·제도 설명 AI
사용자 질문: "전입신고가 뭐예요?"
기존 검색: 정부 사이트 긴 글
서비스:
```
전입신고는
"내가 새로운 주소로 이사했다는 것을 정부에 알리는 절차"입니다.

왜 필요한가요?

- 주민등록 주소 변경
- 학교 배정
- 각종 행정 서비스 이용

해야 하는 경우:
이사 후 14일 이내

필요한 것:
- 신분증
- 새로운 주소
```
### 3. 상황 기반 법률/제도 안내
단순 Q&A 말고:
상황 입력
```
나는 한국에 온 지 3개월 됐고,
아이 학교를 보내고 싶어요.
```
AI
```
현재 필요한 절차:

1단계
외국인 등록 확인

2단계
학교 입학 관련 서류 준비

3단계
교육청 문의

필요 서류:
- 가족관계 증명
- 체류 관련 서류
```
### 4. 쉬운 말 변환
한국 행정문 특유의 문제
원문: "미이행 시 불이익이 발생할 수 있습니다."
AI: "정해진 기간 안에 하지 않으면 문제가 생길 수 있습니다."

### 5. 다국어 지원
지원:
* 한국어
* 영어
* 중국어
* 베트남어
* 일본어
